# Ejecutar en desarrollo
## 1. Clonar el Repositorio

## 2. Instalar Nest CLI
```
npm  i -g @nestjs/cli
```

## Levantar la base de datos 

```
docker-compose up -d
```

## 2. Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```


## Stack Usado 
1. Mongodb
2. Nestjs
